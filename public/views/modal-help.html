Sender Helper

There are some wildcard supported in Fix Client, it is wrapped by %% %%

%%guid%%

Used to generate an unique GUID string

%%now%%

Used to generate a UTC now time, it supports add/minus time, for example: %%now+1d%% means add 1 day to current now time.

%%timestamp%%

Used to generate the unix epoch time stamp.

%%randomdouble:A:B%%

Used to generate a random float number between A and B, for example: %%randomdouble:1.0:10.0%%, it will generate a random number between 1.0 and 10.0.

%%randomnumber:length%%

Used to generate a random number, for example: %%randomnumber:3%%, it might generate a number 143.

%%randomstring:length:<seed>%%

Used to generate a random string, for example: %%randomstring:5%%, it might generate a random string "d4nx2" or %%randomstring:3:abc%%, it might generate a random string "bac".

%%exp[<expression>]%%

Used to generate some value which via executing the expressions, for example: %%exp[2+2]%%, it will get the value equals to 4.
